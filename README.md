# TanStack Mini Admin Dashboard

A premium, modern client-side administration control panel built to demonstrate state-of-the-art frontend patterns with **TanStack Router**, **TanStack Query**, **TypeScript**, and **Tailwind CSS**.

---

## 🚀 Key Features

- **Fake Authentication Flow**: Fully secure-looking login view featuring form validation, interactive inputs, and automatic redirection of unauthenticated users.
- **Interactive Dashboard Overview**: High-fidelity cards showing key metrics (Total Users, Total Tasks, Completion rates with dynamic progress bars) and summary feeds of active tasks & members.
- **Users Management Module**: 
  - Complete CRUD functionality persisting changes directly to `localStorage`.
  - Advanced client-side search by name/email.
  - Filter option by roles (Admin, Manager, Member).
  - Premium confirmation dialogs for profile deletions.
- **Kanban Tasks Board**: 
  - Status columns (`TODO`, `IN_PROGRESS`, `DONE`) displaying assignee metadata, due dates, and priority indicators (High, Medium, Low).
  - Move-direction controls for updating statuses.
  - Full Task CRUD.
- **Responsive Layout**: Designed from scratch to support mobile, tablet, and desktop screens with a side navigation menu.

---

## 🛠️ Technology Stack

- **Framework**: React + Vite + TypeScript
- **Routing**: `@tanstack/react-router` (File-based routing & type-safe link validation)
- **Data Caching & Mutations**: `@tanstack/react-query` (Configured for efficient local query invalidation)
- **Styling**: Tailwind CSS v4.0 (Utilizing CSS-first configuration and modern CSS features)
- **Icons**: `lucide-react` (Crisp SVG vector assets)

## ⚙️ Getting Started

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

### 2. Clone & Install Dependencies
Navigate to the project folder and run:
```bash
npm install
```

### 3. Setup Environment Variables
Duplicate the template:
```bash
cp .env.example .env
```

### 4. Run Development Server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your web browser.
